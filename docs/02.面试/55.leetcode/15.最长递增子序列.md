# 最长递增子序列

最长递增子序列问题是在给定序列中找到一个子序列，其中子序列的元素按从低到高的顺序排列，并且子序列尽可能长。该子序列不一定是连续的或唯一的。
<!-- more -->

示例：
```js
// 0, 8, 4, 12, 2, 10, 6, 14, 1, 9, 5, 13, 3, 11, 7, 15

// 最长的递增子序列是
0, 2, 6, 9, 11, 15

// 或者（长度相等）
0, 4, 6, 9, 11, 15 or
0, 2, 6, 9, 13, 15 or
0, 4, 6, 9, 13, 15
```
