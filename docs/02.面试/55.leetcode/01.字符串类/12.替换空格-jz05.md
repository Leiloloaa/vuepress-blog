# 替换空格-jz05

> jz 表示 🗡☞ 剑指 offer

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

示例 1：
输入：s = "We are happy."
输出："We%20are%20happy."

限制：
0 <= s 的长度 <= 10000

<!-- more -->

## 解题思路

- 极致的优化题目，就是不用额外的辅助空间
- 题目也类似于匹配问题，匹配空格，然后转换成 %20
- 先获取空格的数量 k，然后再增加 2k 个长度
- 再双指针遍历

## 答案

```js
function replaceSpace(s) {
    let arr = s.split('');
    let count = 0; // 空格数
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === ' ') {
            ++count;
        }
    }
    // 双指针
    // 从后往前移动
    // 长度为 8 的话 是要从 位置 7 开始
    let l = arr.length - 1;
    let r = arr.length + count * 2 - 1;
    // 循环到 第一个 位置
    while (l >= 0) {
        if (arr[l] === ' ') {
            arr[r--] = '0';
            arr[r--] = '2';
            arr[r--] = '%';
            l--;
        } else {
            arr[r--] = arr[l--];
        }
    }
    return arr.join('');
}

console.log(replaceSpace('d ca bab c'));
```
