# vite

- Vite 专注于提高开发环境的速度，上线时 Vite 还是使用内置的 rollup 实现文件打包
- Vite 使用了 esbuild 来处理 javascript 文件，这也是 vite 速度快的主要原因之一
- Vite 基于浏览器自己的 module import 功能，可以做到首页文件按需节加载
- 针对文件过多的库，vite 内置了预打包的功能避免过多的 http 请求
