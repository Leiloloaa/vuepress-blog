# 查找唯一编号

> 有一个带有一些数字的数组。除一个以外，所有数字均相等。尝试找到它！

> findUniq([ 1, 1, 1, 2, 1, 1 ]) === 2 findUniq([ 0, 0, 0.55, 0, 0 ]) === 0.55

```javascript
function findUniq(arr) {
  return  (arr.filter((val,i,arr) => arr.indexOf(val) == i && arr.lastIndexOf(val) == i))[0]
}

```
