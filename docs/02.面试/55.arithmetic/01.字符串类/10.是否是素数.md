# 是否是素数

> 素数（或素数）是大于1的自然数，除1及其本身外，没有除数。

```javascript

function isPrime(num) {
  //素数 除了 1 和 本身 没有 除数
  if(num <= 1 ) return false;
  for(let i = 2 ; i < num; i++){
    if(num % i == 0){
      return false
    }
  }
  return true
}
```
