# 分割字符串

> 将字符串分成两个字符对。如果字符串中包含奇数个字符，则应使用下划线（_）替换最后一对中丢失的第二个字符
> 例如 solution('abc') // should return ['ab', 'c_'] solution('abcdef') // should return ['ab', 'cd', 'ef']

```javascript
function solution(str){
   if(!str) return [];
   if(str.length % 2 !== 0){
     str = str + '_'
   }
   let i = 0
   let result = []
   while(i<str.length){
     result.push(str[i] + str[i+1])
     i += 2
   }
  return result
}
```
