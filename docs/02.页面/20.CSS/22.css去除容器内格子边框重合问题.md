# css 去除容器内格子边框重合问题

## table

设置 css 属性 border-collapse:collapse。

## div

使用伪类

```css
/*父元素拥有上边框*/

.item {
    border-top: 1px solid #dfdfdf;
}

/*最后一个元素及每行末元素拥有右边框*/

.item :nth-child(n) {
    border-right: 1px solid #dfdfdf;
}

.item :last-child {
    border-right: none;
}
```
